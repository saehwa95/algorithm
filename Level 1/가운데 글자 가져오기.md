# 가운데 글자 가져오기

## 1. 문제설명
 + 단어 s의 가운데 글자를 반환하는 함수, solution 만들기
 + 단어의 길이가 짝수라면 가운데 두글자를 반환
  
## 2. 제한조건
 + s는 길이가 1이상, 100이하인 string
  
## 3. 기본코드
```JavaScript
function solution(s) {
    var answer = '';
    return answer;
}
```

## 4. 문제풀이
```JavaScript
function solution(s) {
    var answer = '';
    if (s.length % 2 == 0) {
        answer = s[s.length / 2 - 1] + s[s.length/2] 
    } else {
        answer = s[Math.floor(s.length / 2)]
    }
    return answer;
}
```
