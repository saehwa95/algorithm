# 두 정수 사이의 합

## 1. 문제설명
 + 두 정수 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수
 
## 2. 제한조건
 + a와 b가 같은 경우는 둘 중 아무 수나 리턴
 + a와 b의 대소관계는 정해져있지 않음
  
## 3. 기본코드
```
function solution(a, b) {
    var answer = 0;
    return answer;
}
```

## 4. 문제풀이
```
function solution(a, b) {
    var answer = 0;
    if(a>=b){
        for (let i = b; i <= a; i++) {
            answer += i;
        } 
    } else {
        for (let j = a; j <= b; j++) {
            answer += j;
        }
    }
    return answer;
}
```
