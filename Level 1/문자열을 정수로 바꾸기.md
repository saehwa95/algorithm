# 문자열을 정수로 바꾸기

## 1. 문제설명
 + 문자열 s 를 숫자로 반환하는 함수, solution 완성하기
  
## 2. 제한조건
 + s의 길이는 1이상 5이하
 + s의 맨앞에는 부호(+, -)가 올 수 있음
 + s는 부호와 숫자로만 이루어져있음
 + s는 "0"으로 시작하지 않음
  
## 3. 기본코드
 ``` JavaScript
function solution(s) {
    var answer = 0;
    return answer;
}
```
 
## 4. 문제풀이
  ``` JavaScript
function solution(s) {
    return parseInt(s);
}
```
